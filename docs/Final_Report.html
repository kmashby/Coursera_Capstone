Battle of the Neighborhoods: Chicago Pet Store¶

Coursera Applied Data Science Capstone Assignment¶

Introduction¶

A client is looking for the perfect location to open a new pet store in Chicago, IL. We will help out client narrow their search by identifying neighborhoods that are similar to the neighborhoods that already have pet stores as these may offer ideal locations for a new pet store.

To accomplish this goal, I will use the Foursquare API to download venue data for each neighborhood in Chicago and then cluster the neighborhood data using k-means. Neighborhoods within the same cluster as those with existing pet stores could provide the best location for a new pet store.

Data¶

Chicago Neighborhoods¶

Chicago neighborhood names and boundary data can be downloaded from the Chicago city website (https://data.cityofchicago.org/resource/igwz-8jzy.json). The boundaries information is provided as a multipolygon with multiple latitude and longitude points for each neighborhood. A centroid point can be calculated from the boundary points and the latitude and longitude of the centroid point can be used as the location for the neighborhood in the Foursquare API query.

First, download the data from the city of Chicago website:

def calculateCentroid(data):
    
    x, y = zip(*data[0]['coordinates'][0][0])
    lat = sum(x) / len(x)
    lng = sum(y) / len(y)
    return(lat, lng)
    
centroid = neighborhoods[["the_geom"]].apply(calculateCentroid, axis=1, result_type='expand')

neighborhoods["lng"] = centroid[0]
neighborhoods["lat"] = centroid[1]

neighborhoods.drop(["the_geom"], axis=1, inplace=True)

Methodology¶

Before beginning analysis I will first preprocess the data and remove neighborhoods with very few venues as well as venues that occur only rarely.

Once the data is filtered, we can calculate the mean occurance of the venues in each neighborhood. Since the data is still very sparse, we can use PCA to reduce the dimensionality. Then we use the reduced dataset to cluster the neighborhoods using k-means clustering. The number of clusters is determined using the silhouette score and plots to visualize the seperation of the clusters.

After the neighborhoods are clustered, we can identify the neighborhoods that are in the same cluster or are simialr to the neighborhoods with existing pet stores. The neighborhoods that are similar to those that already have a pet store are the best neighborhoods for a new pet store.

Discussion¶

Chicago is known for its deep dish pizza and its magnificant parks nestled between skyscrappers and Lake Michigan so it is no suprise that pizza place and park are the two most common in the cluster 0 since its mostly near the shoreline and in areas that are popular with tourists. Most of the pet shops are in cluster 3.

One limitation of this analysis is that the k-means results were quite varied depending on the preprocessing approach. Without reducing dimensions using PCA, the majority of results all clustered together. It would be interesting to try a method that incorporates features selection, such as a boosted tree.

In addition, there may also be a better was to link the venues to their respective neighborhoods. In this analysis, I downloaded all venues within a certain radius of the neighborhood centroid point. However; this technique is not exact as some neighborhoods are more dense and some are more spread out. If you choose a radius that it too large means, you may end up with duplicated venues from overlapping results, but if you select a radius that is too small, you will not download all the venues for neighborhoods that are more spread out. A better technique might be to use a large radius from the Chicago latitude and longitude and then assign results to neighborhoods based on latitude and longitude data. This would have been possible with the boundary data provided in the Chicago data.

Conclusion¶

The best neighborhood for a new pet store is one of the neighborhoods in cluster 3. These are most similar neighborhoods that already have pet stores. We would recommend that our client choose a neighborhood in cluster 3 that does not already have a pet store.
